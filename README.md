проект Для Яндекс.Лицей - WebServer

Электронный читательский билет для посетителей библиотеки

Запуск сервера:
cgi-bin/library.py


Раздел для Администрирования:
http://127.0.0.1:8080/admin

На этой странице отображается список читателей библиотеки (логин, пароль и количество книг на руках у пользователя)
Также можно добавлять новых пользователей и новые книги в библиотеку

Раздел для читателей:
http://127.0.0.1:8080/

После авторизации читатель видит экземпляры книг которые у него на руках.
Также он может взять книгу из библиотеки. В соответствующем разделе он видит список доступных книг и может выбрать интересующую его книгу.
После выбора новой книги, она появляется в списке его книг, а количество экземпляров книг доступных для других пользователей уменьшается на единицу.
Если пользователь возвращает книгу в библиотеку (ссылка Вернуть экземпляр в библиотеку) - данный экземпляр пропадает из списка на странице пользователя, а количество доступных экземпляров в библиотеке увеличивается на единицу.
